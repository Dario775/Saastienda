# Métricas Clave de Rendimiento (KPIs)

## 1. Métricas Financieras y Comerciales

| KPI | Descripción | Por qué es Vital |
| :--- | :--- | :--- |
| **MRR (Monthly Recurring Revenue)** | Ingresos Recurrentes Mensuales. La suma de todas las suscripciones activas normalizadas a un valor mensual. | Es el pulso financiero del SaaS. Indica la estabilidad y el crecimiento predecible del flujo de caja. |
| **Churn Rate (Tasa de Cancelación)** | Porcentaje de clientes que cancelan su suscripción en un periodo determinado. | Mide la retención y satisfacción. Un churn alto mata el crecimiento, sin importar cuántos clientes nuevos se adquieran. |
| **GMV (Gross Merchandise Value)** | Valor Bruto de Mercancía. El valor total de las ventas realizadas por las tiendas alojadas en la plataforma. | Aunque no cobremos comisión, el GMV demuestra el éxito de nuestros clientes. Si ellos venden, se quedan. |

## 2. Métricas Técnicas y Operacionales

| KPI | Descripción | Por qué es Vital |
| :--- | :--- | :--- |
| **Uptime / Disponibilidad** | Porcentaje de tiempo que la plataforma está accesible y funcional (Meta: 99.9%+). | La confianza es la moneda del e-commerce. Si la tienda no carga, el cliente pierde ventas y nosotros perdemos al cliente. |
| **Latencia de API (P95)** | Tiempo que tarda el servidor en procesar una solicitud (medido en el percentil 95 para capturar casos lentos). | La velocidad es conversión. Una API lenta frustra a los compradores finales y afecta el SEO de las tiendas. |
